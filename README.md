
## Description 📚
 Food Recipe  app. This app uses the Redux for state management,along with mui. 
## Installation 💡 
 npm install



## Live Demo 🌐 :

https://foodieshub.vercel.app


## Install 📔

npm start

Runs the app in the development mode.
Open http://localhost:3000


npm run build

Builds the app for production to the build folder.
